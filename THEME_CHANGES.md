# Cambios de Tema - El Hogar Macorisano

## Resumen
La página ha sido transformada de "Clínica Santiago Apóstol" a "El Hogar Macorisano" con un tema elegante, moderno y con esquinas cuadradas.

## Cambios Principales

### 1. **Paleta de Colores**
- **Anterior**: Verde médico
- **Nuevo**: Naranja cálido (Warm Orange #D97834 - RGB: 217, 120, 52)
- **Secundario**: Ámbar suave para acentos
- **Fondos**: Tonos cálidos y acogedores

### 2. **Diseño Geométrico**
- **Border Radius**: Reducido de 0.75rem a 0.5rem para aspecto más cuadrado y moderno
- **Esquinas**: Menos redondeadas en tarjetas, botones y elementos
- **Cards**: Estilo limpio y minimalista con bordes definidos

### 3. **Secciones Rediseñadas**

#### Anterior → Nuevo
- Hero: Atención médica → Hogar y muebles
- Servicios → Catálogo de productos
- Gerencia → Ofertas y Promociones
- Testimonios → Clientes satisfechos
- Ubicación → Sucursales con Google Maps

### 4. **Contenido**

#### Datos Principales
- **Empresa**: El Hogar Macorisano
- **Tagline**: "Tu casa, como siempre la soñaste"
- **Ubicación Principal**: San Francisco de Macorís
- **Sucursales**: San Francisco de Macorís, Cotuí, Gaspar Hernández, Moca, Nagua, Tenares

#### Categorías de Productos
- Muebles de Sala
- Electrodomésticos
- Artículos para Cocina
- Muebles Dormitorio
- Decoración
- Promociones especiales

### 5. **Iconografía**
- ✓ Home (Logo del hogar)
- ✓ Sofa (Muebles)
- ✓ Refrigerator (Electrodomésticos)
- ✓ UtensilsCrossed (Cocina)
- ✓ Sparkles (Decoración)
- ✓ Star (Calificaciones)
- ✓ MapPin (Ubicaciones)
- ✓ Instagram (Redes sociales)

### 6. **Integración de Medios**
- **Instagram**: @elhogarmacorisano (https://www.instagram.com/elhogarmacorisano/)
- **Imágenes**: Se utilizan las fotos de la carpeta `/public/`
- **Google Maps**: Integración de ubicaciones con navegación directa

### 7. **Secciones Principales**
1. **Header/Navbar**: Navegación elegante
2. **Hero**: Presentación principal con CTA
3. **Catálogo**: Categorías de productos
4. **Quiénes Somos**: Historia y valores
5. **Ofertas**: Promotions con Instagram embed
6. **Clientes**: Testimonios
7. **Confianza**: Estadísticas de satisfacción
8. **Sucursales**: Mapa interactivo y tarjetas de ubicaciones
9. **Footer**: Información y enlaces

## Notas Técnicas

- **Framework**: React + Vite
- **Animaciones**: Framer Motion
- **Estilos**: Tailwind CSS
- **Componentes**: Shadcn/ui
- **Colores**: Sistema HSL con variables CSS
- **Border Radius**: 0.5rem (8px) para aspecto cuadrado
